garmin變更管理流程優化

1.問題確認
變更管理的制度於2021年8月才開始實施，因當時都未實施管理但工安事故及設備問題頻傳，礙於時間壓力設計出簡易流程，希望能在設備用於生產前經歷完整安全等一系列評估，後來實施一段期間後發現卻無法大幅改善現況，主要問題是時程拖很久才完成。
於是瞭解現況並經查核後，因目前申請系統紀錄的數據量太少，建議在填寫時新增欄位：每個階段完成的時間記錄、超時原因(採用順序資料收集)。
